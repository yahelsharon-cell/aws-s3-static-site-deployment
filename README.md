# AWS S3 Static Website Deployment

## Project Description
This project demonstrates the deployment of a static website using AWS cloud infrastructure.
*Note: The HTML content serves as a template to demonstrate the cloud configuration and deployment process.*

## What I Did (Infrastructure & DevOps)
* **Cloud Storage:** Created and configured an Amazon S3 Bucket.
* **Permissions:** Managed **Block Public Access** settings and wrote a **Bucket Policy (JSON)** to allow public read access securely.
* **Configuration:** Enabled Static Website Hosting and configured the index document.
* **Troubleshooting:** Fixed MIME type/metadata issues directly in the AWS Console to ensure correct browser rendering.

## Tech Stack
* AWS S3
* Cloud Security (IAM Policies)
* HTML5
